optimal deadlines

模型简介

本文所采用的模型源自李浩、孙宁等学者发表于TE的《Optimal Deadline for Agreements》，该模型聚焦信息与谈判的互动机制，将带最后期限的双边谈判构建为对称连续时间 “消耗战” 框架。
在该谈判场景中，双方分别提出一个方案，每个方案均包含 “共同价值”（由双方共享）与 “私人价值”（仅对提出方有利）两个维度。谈判过程中，任一时刻双方均面临两种策略选择：一是坚持自身提出的方案，二是向对方的方案让步。若双方陷入僵持，需共同承担由谈判延迟产生的成本，这一僵持状态将持续至以下两种结果之一出现：要么双方达成协议并立即执行某一方案，要么谈判期限届满，此时由随机机制（公平硬币投掷）决定最终执行的方案。
信息结构方面，谈判双方仅知晓自身所提方案的质量水平，对对方方案质量完全未知。进一步分析显示：当质量差异高于私人收益时，若高质量方案方与低质量方案方实现信息共享，双方将一致同意选择高质量方案；但若双方均为低质量方案方，即便完全知情，仍会因私人收益的存在而坚持各自方案，导致分歧。这一特征使得谈判期限具备 “福利效应”（源于协议达成的可能性），同时也使信息共享产生 “高昂成本”（源于分歧持续的可能性）。

具体设定

本模型为对称双人方案选择模型，核心设定如下：
谈判双方需共同确定最终执行的方案，若无法达成共识则进入下一期谈判。双方提出的方案均由 “共同价值部分” 与 “私人价值部分” 构成：
共同价值部分：仅存在两种状态，分别为对双方均产生同等收益的较低水平$\upsilon_L$与较高水平$\upsilon_H$，满足$\upsilon_H > \upsilon_L > 0$；
私人价值部分：为固定收益$\beta$（$\beta > 0$），且仅由方案提出方获得。基于此，定义某一谈判方的 “偏好方案” 为：能为该方带来私人价值$\beta$的方案（即自身提出的方案）。
收益计算规则为：若谈判方最终选择自身偏好方案，其总收益等于该方案的共同价值与私人价值$\beta$之和；若选择对方偏好方案（即让步），其总收益仅等于该方案的共同价值（无法获得私人价值$\beta$）。

假设

为简化分析并聚焦核心机制，模型附加以下假设（注：此类假设为模型设定的必要组成部分，旨在方便分析）：
让步激励假设：$\upsilon_H - \beta > \upsilon_L$。该不等式确保当低质量方案提供者（即自身方案共同价值为$\upsilon_L$的谈判方）面对高质量方案提供者（即对方方案共同价值为$\upsilon_H$的谈判方）时，存在向对方方案让步的激励 —— 此时低质量方案方选择让步后的收益（$\upsilon_H$）高于坚持自身方案的收益（$\upsilon_L + \beta$）。
类型组合假设：谈判双方的方案质量仅存在两种可能组合：一是 “一高一低”（一方为高质量方案，另一方为低质量方案），二是 “双低”（双方均为低质量方案）。这一假设隐含的信息推断逻辑为：若某谈判方知晓自身为高质量方案方，则可直接确定对方必为低质量方案方，此时该方的最优策略为坚持自身偏好方案；若某谈判方知晓自身为低质量方案方，则无法确定对方类型 —— 对方可能是高质量方案方（此时属于 “可达成协议的状态”），也可能是低质量方案方（此时属于 “易产生分歧的状态”）。
定义$\gamma_0$（$0 < \gamma_0 < 1$）为低类型谈判方（自身方案共同价值为$\upsilon_L$）的初始信念，即其认为当前处于分歧状态（对方亦为低类型）的概率，且$\gamma_0$为双方共同知识。模型中谈判双方类型组合存在三种可能状态，基于对称性设$\text{Pr}(\text{任一玩家为低类型}) = P$，结合贝叶斯概率法则：分歧状态（双方均低）概率为$\text{Pr}(\text{低}_1,\text{低}_2) = P \cdot \gamma_0$（因低类型玩家认为对方低的概率为$\gamma_0$）；两类协议状态（低高、高低）概率均为$P \cdot (1-\gamma_0)$（因低类型玩家认为对方高的概率为$1-\gamma_0$）。由总概率为1，即$P \cdot \gamma_0 + 2 \cdot P \cdot (1-\gamma_0) = 1$，解得$P = \frac{1}{2 - \gamma_0}$。进而推得：分歧状态先验概率为$\frac{\gamma_0}{2 - \gamma_0}$，某一类协议状态先验概率为$\frac{1 - \gamma_0}{2 - \gamma_0}$。
该谈判博弈在连续时间内以重复提案形式进行，从$t = 0$开始直至最后期限$T$（$T$可为有限值或无限大），任一时刻$t$（$0 \leq t < T$）双方需同时提案，若提案一致则博弈立即结束并执行该方案，若不一致则继续僵持，若持续至$T$仍未达成一致则通过公平抛币（正反面概率均为$0.5$）决定执行方案，且博弈过程中双方均需承担延迟流动损失，损失率为$\delta$（$\delta > 0$），损失随时间连续累积。

静态分析

如果双方直接到达谈判期限，即T=0时，我们来分析低类型的行为，当他选择让步时，获得的收益为$\gamma_0V_L+(1-\gamma_0)V_h$,当他选择坚持自己的选择时，得到的收益为$\gamma_0(V_L+\frac{\beta}{2})+(1-\gamma_0)\frac{V_h+V_L+\beta}{2}$,两者做差，我们得到当若低类型参与者的初始信念满足以下条件，则其会严格偏好自己最倾向的选择：
$$\gamma_0 > \gamma^* \equiv \frac{\upsilon_H - \upsilon_L - \beta}{\upsilon_H - \upsilon_L}$$
若$\gamma_0$小于$\gamma^*,$则低类型会选择坚持，如果$\gamma_0$等于$\gamma^*$,则低类型对两种方案无差异。由此，我们得到了低类型的策略，类似的，我们可以推出高类型知道低类型的策略后，仍然严格偏好于坚持自己的策略。

若分别用 $U^0_L(\gamma)$ 和 $U^0_H(\gamma)$ 表示低类型与高类型参与者的均衡收益，则有：

$$
U^0_L(\gamma) =
\begin{cases}
\frac{1}{2}\gamma(2\upsilon_L + \beta) + (1 - \gamma)\upsilon_H, \gamma \in [0, \gamma^*),\
\frac{1}{2}\gamma(2\upsilon_L + \beta) + \frac{1}{2}(1 - \gamma)(\upsilon_H + \upsilon_L + \beta), \gamma \in (\gamma^*, 1]
\end{cases}
$$
且满足：
$U^0_L(\gamma^*) \in \left[ \tfrac{1}{2}\gamma^*(2\upsilon_L + \beta) + \tfrac{1}{2}(1 - \gamma^*)(\upsilon_H + \upsilon_L + \beta),\ \tfrac{1}{2}\gamma^*(2\upsilon_L + \beta) + (1 - \gamma^*)\upsilon_H \right]$

同时，高类型参与者的均衡收益为：

$U^0_H(\gamma) =
\begin{cases}
\upsilon_H + \beta & \text{若 } \gamma \in [0, \gamma^*), \
\in \left[ \tfrac{1}{2}(\upsilon_H + \upsilon_L + \beta),\ \upsilon_H + \beta \right] & \text{若 } \gamma = \gamma^*, \
\tfrac{1}{2}(\upsilon_H + \upsilon_L + \beta) & \text{若 } \gamma \in (\gamma^*, 1].
\end{cases}$

由于模型具有对称性，分歧状态下的任何结果均满足帕累托效率（Pareto-efficient）。因此，当 $\gamma \in [0, \gamma^*)$时，高类型与低类型参与者均能获得其“最优期望收益”（first best expected payoffs）。然而，当 $\gamma \in (\gamma^*, 1]$时，均衡结果是无效率的——因为若低类型参与者同意选择对手偏好的方案，而非通过掷硬币决定，两类参与者的期望收益均会提升。
